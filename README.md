# AI Powered Resume Screening App (MERN + AI)

## Overview
This is an AI-powered resume screening web application built using the MERN stack.
Users can upload their resume (PDF), and the system analyzes it against a job description
to generate an ATS-style score and feedback.

## Tech Stack
- Frontend: React.js, Context API, HOC
- Backend: Node.js, Express.js
- Database: MongoDB
- AI/NLP: Resume text parsing & scoring logic
- File Upload: Multer
- PDF Parsing: pdf-parse

## Key Features
- User authentication (Login / Logout)
- Resume upload (PDF)
- Resume parsing & keyword extraction
- ATS score generation
- Admin & User dashboards
- Secure API-based architecture
